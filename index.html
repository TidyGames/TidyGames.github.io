  • Hippity Hoppity

    The first commercial game that Tidy Games released for Android.
    A simple game created in a short time frame, this game wasn't intended as a debut, but rather to give the team experience in the processes required when releasing a game.

    When producing Hippity Hoppity we worked with freelance artists for both the music and the art.

    Updates to the game are on hold while we work on other projects, however we hope to add more skins, improve the quality of our tutorial and add a global leaderboard.

    Saucer Sling

    Saucer Sling was originally the first game created by "Team Tidy" for the Global Game Jam 2018

    Created in Unity, this small gravity puzzle game is now being re-created by Tidy Games with our much improved skill set.

    The core game is an uncontrollable ship which needs to get to the exit wormhole. There are many obstacles and tools for the player to use to help them reach the end but you can't ever directly control your 'saucer'

    An in-game level editor is in the works so that players can create their own levels and play other fan-made puzzles all from within the game.

    AR Maths - Experimental Project

    During our second year of university, Tidy Games worked with a client for one of our assignments. Our brief was to create an Augmented Reality (AR) program/game that could be used to help children learn maths.

    We continued working with the client after the assignment and created a prototype that extended the use of the AR to create a menu system that would allow not only a simple maths game, but also additional learning resources to be added.

    Unfortunately the project never recieved funding and has been put to one side until further notice. The experience was still incredibly useful to the team as we gained valuable experience working with a client and interpreting their requests as well as learning about emerging technologies and how we can use them in game making and gamification.

    UWE E&T Degree show 2019 Demo

    The Turing Test

    The Turing Test was a coursework project created by Tidy Games in their 2nd year of university.

    Written using C++ the game had simple stealth mechanics with conversation elements. The goal of the game was to escape the facility without the guards realising you were an irtificial intelligence.

    In this project we also created a simple level parser to create tile based maps from a grid of numbers.
